🌐 Hosting a Web Server on AWS Cloud

This project demonstrates how to build a secure and scalable cloud network and deploy a website on AWS.

  1. 🏛 Step 1 – Create a VPC
  2. 🌐 Step 2 – Create a Public Subnet
  3. 🔗 Step 3 – Attach Internet Gateway (IGW)
  4. 🛣 Step 4 – Create Route Table for Public Traffic
  5. 🔐 Step 5 – Create Security Group (Firewall)
  6. 🖥 Step 6 – Launch Virtual Machine (EC2)
  7. 🌍 Step 7 – Connect to Your VM
  8. 🛠 Step 8 – Install Web Server (Nginx)
  9. 🌐 Step 9 – Deploy Your Website

Outcome: A fully functional website hosted on AWS EC2, accessible globally through your public IP or domain name.
