Static Website Hosting using AWS S3

I have successfully hosted a static website using **Amazon S3** in a few simple steps!

With just a few configurations, you can quickly deploy your site and make it available to the world.

It's as simple as:

  • ✔ Create an S3 bucket
  • ✔ Enable static website hosting
  • ✔ Upload your website files
  • ✔ Set the correct permissions
